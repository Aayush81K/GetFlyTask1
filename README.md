# Notification and FAQ Page

This project is a simple and colorful React application that displays the latest updates and frequently asked questions (FAQ) for students. It includes a navigation bar, a notifications section, and a FAQ section, all styled with vibrant and interactive elements.

## Features
- **Navigation Bar**: Contains a logo on the left-hand side and navigation links on the right-hand side.
- **Notifications Section**: Displays the latest updates regarding posted seminars with each card containing a title and date of the update.
- **FAQ Section**: Shows the frequently asked questions with an option to raise new questions.

## Technologies Used
- **React**: For building the user interface.
- **CSS**: For styling the application with a colorful and engaging design.

## Installation
1. Clone the repository:
    ```sh
    git clone https://github.com/Aayush81K/GetFlyTask1
    ```
2. Navigate to the project directory:
    ```sh
    cd GetFlyTask1
    ```
3. Install the necessary dependencies:
    ```sh
    npm install
    ```

## Usage
1. Start the development server:
    ```sh
    npm start
    ```
2. Open your browser and navigate to view the application.

## Components

### NavBar
The navigation bar component containing the logo and navigation links.

### Notifications
The main container for displaying the notification cards.

#### NotificationCard
A component that represents an individual notification update with a title and date.

### FAQs
The main container for displaying the FAQ cards and a button to add new questions.

#### FAQCard
A component that represents an individual FAQ item with a question and answer.
